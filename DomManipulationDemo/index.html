Todo List

Welcome to the Todo List app! This app demonstrates how we can use JavaScript and jQuery to manipulate the DOM (Document Object Model) to dynamically add, remove, and modify elements.

Adding a New Task

Enter a task in the input field below and click the "Add Task" button to add it to the list. We are selecting the DOM elements using JavaScript and jQuery here.

Your Tasks

Here are the tasks you've added. You can click on a task to mark it as completed, or click the "Remove" button to delete a task.
